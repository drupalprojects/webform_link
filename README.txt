Description
-----------
Webform Link creates a webform component that lets the user enter a URL.
The module verifies that a valid URL is entered and displays the result as a link.

Requirements
------------
drupal 7.x
webform

Installation
------------
1. Copy the entire module directory the Drupal sites/all/modules directory.

2. Login as an administrator. Enable the module in the "Administer" -> "Modules"

3. Create a webform node at node/add/webform.

4. Add a component named "Link".
